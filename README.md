Getting Started
Prerequisites
Node.js (only if you want to use SCSS compilation or JavaScript build tools)
npm (optional, for SCSS and build tools)
Installation
Clone the repository

bash
git clone https://github.com/Abishek7952/Architecture-Webpage.git
cd Architecture-Webpage
Open in your browser
Simply open index.html in your preferred browser.

(Optional) SCSS Compilation
If you wish to modify SCSS files:

Install dependencies:
bash
npm install
Compile SCSS:
bash
npm run build
Project Structure
plaintext
.
├── css/           # Compiled CSS files
├── scss/          # Source SCSS files
├── js/            # JavaScript files
├── index.html     # Main HTML file
└── assets/        # Images and other assets
Customization
Colors & Typography: Edit the SCSS files in the /scss directory.
Content: Update HTML in index.html or add new sections as needed.
Images: Place your images in the /assets directory.
